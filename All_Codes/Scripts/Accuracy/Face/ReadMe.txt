Purpose:
	To find the accuracy of Face Detection
How To Run:
	Copy and paste the contents of the "XML_Parser" folder in the "All_XML's" folder.
	Run the command "python xmlparser.py"
	Copy and paste the contents of the "Accuracy_Scripts" folder in the "Images".
	Run the command "python faceonlycopy.py"
Output:
	xmlparser.py: This file outputs a csv data in the file output_xml.txt which has the format file_name,bounding_box_size,illumination condition
	faceonlycopy.py: This file outputs the data in a txt file which contains the information about in which all files faces are detected and in which all files faces are not detected.
